◆ matrix_is_symmetric()

bool matrix_is_symmetric ( Matrix m,
double tolerance )

Проверяет, является ли матрица симметричной

Parameters
mМатрица для проверки
toleranceДопустимая погрешность для сравнения чисел с плавающей точкой
Returns
true если матрица квадратная и симметричная, false в противном случае
Note
Матрица считается симметричной, если она равна своей транспонированной версии: A[i][j] == A[j][i] для всех i, j
Для неквадратных матриц всегда возвращается false
